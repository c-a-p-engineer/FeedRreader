# FeedRreader
RSS Feed Reader
# Install

# Usage

Feed URL Setting
```ini:setting.ini
[RSS URLS]
urls[] = "https://news.yahoo.co.jp/rss/topics/it.xml"
urls[] = "https://www.soumu.go.jp/news.rdf"
```

Feed Result HTML Create
```bash
php feed.php
```
